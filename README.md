Data Science Notebook
Project: Alcohol Consumption, Happiness, and GDP Analysis
Overview
This project explores the relationship between alcohol consumption, happiness levels, and GDP per capita across countries.
The goal is to identify patterns and correlations between economic conditions, well-being, and lifestyle factors using data analysis techniques.
________________________________________
Dataset Description
The analysis combines datasets containing:
•	Alcohol consumption per capita
•	Happiness scores (e.g. World Happiness Report)
•	GDP per capita
These datasets are merged and analyzed to explore cross-country relationships.
________________________________________
Key Questions
•	Is there a relationship between GDP per capita and happiness?
•	Does alcohol consumption correlate with happiness?
•	How does economic development influence lifestyle and well-being?

Analysis Steps
•	Data cleaning and preprocessing
•	Merging multiple datasets
•	Exploratory Data Analysis (EDA)
•	Correlation analysis
•	Data visualization (scatter plots, trends)
________________________________________
Key Insights (to be updated)
•	Higher GDP per capita generally correlates with higher happiness scores
•	Alcohol consumption shows varying patterns depending on region and income level
•	Relationships are not strictly linear and may depend on additional factors
